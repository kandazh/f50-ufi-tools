    Advanced Features
    ⚠️ Warning: This feature is for advanced users. Regular users should not attempt — may cause device malfunction, data loss, or unrecoverable issues.
    Note: Do not disable File Sharing after enabling advanced features, or they will stop working.
    If removing advanced features causes errors, run in TTYD: sh /sdcard/hotbox/unlock_samba.sh Reboot to apply.
    Advanced features include:
    1. Optimized Samba directories (SD card, device root)
    2. TTYD with root shell
    3. Block IPv6 access to TTYD/ADB/SMB for security
    4. Scheduled script (runs every ~30s: hotbox_schedule.sh)
    5. Boot script (hotbox_boot.sh on internal storage)
    (Thanks to @QuTick @AiCherub @atwat for the ideas)

Quick Shell (ADB)
Output
$ quick_shell.sh

🔄 Flow: Edit above → Save & Run → saved to device → executed

📁 File: /sdcard/quick_shell.sh

👤 Runs as: shell user (UID 2000) — no root needed

🛠️ Method: ZTE Engineer Mode → DEBUG&LOG → ADB Shell (UI automation)

✅ Use for: granting permissions, enabling WiFi ADB, automating settings — all without root

📖 Example script 
#!/system/bin/sh

# Device info
echo "--- Device ---"
getprop ro.product.model
getprop ro.build.version.release

# Battery status
echo "--- Battery ---"
cat /sys/class/power_supply/battery/capacity
cat /sys/class/power_supply/battery/status

# Network
echo "--- Network ---"
getprop gsm.operator.alpha
getprop gsm.network.type

# List connected WiFi clients
ip neigh show

# Uptime
echo "--- Uptime ---"
uptime

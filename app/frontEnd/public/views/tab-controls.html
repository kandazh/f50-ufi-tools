⚡ Quick Shell (ADB)
Edit /sdcard/quick_shell.sh below, then save or run via ZTE Engineer Mode. Does not require root.
Output
How it works
  1. Create a script at /sdcard/quick_shell.sh
  2. Click Run — the app opens Engineer Mode via ADB
  3. It navigates to DEBUG&LOG → ADB Shell automatically
  4. Your script is typed and executed as the shell user
Use cases:
  • Grant app permissions before root is available
  • Enable ADB over WiFi on a fresh device
  • Automate device settings without root
Example script: 
#!/system/bin/sh
# Enable WiFi ADB
setprop service.adb.tcp.port 5555
stop adbd; start adbd

# Grant UFI Tools permissions
pm grant com.minikano.f50_sms android.permission.READ_SMS
pm grant com.minikano.f50_sms android.permission.SEND_SMS
pm grant com.minikano.f50_sms android.permission.READ_PHONE_STATE

# Keep screen on for 5 min
settings put system screen_off_timeout 300000
